## [0.1.0] - 2025-01-13

### Added

- Initial release of Global News Map
- Interactive Leaflet map with custom news markers
- Country boundary highlighting on hover
- Real-time news simulation with 4-second intervals
- Category filtering (10 categories)
- Statistics panel with category and region breakdown
- News feed sidebar with click-to-locate
- Dark/Light theme toggle
- Responsive design for mobile devices
- Toast notifications for breaking news
- WebSocket hook ready for backend integration

### Technical

- Next.js 16.1.1 with App Router
- React 19 with TypeScript strict mode
- Tailwind CSS 4 for styling
- Leaflet + React Leaflet for maps
- Socket.IO client for real-time capability
